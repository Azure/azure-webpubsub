# Teams-Lite Client

React + TypeScript + Vite frontend for the Teams-Lite chat demo, powered by `@azure/web-pubsub-chat-client`.

## Quick Start

Prereqs: Node.js 18+ and the server running at http://localhost:3000.

```bash
yarn install
yarn dev
```

Open http://localhost:5173

## Scripts

| Command | Description |
|---------|-------------|
| `yarn dev` | Vite dev server |
| `yarn build` | Type check + production build |
| `yarn test` | Vitest + React Testing Library |
| `yarn lint` | ESLint |
