Azure Web PubSub Service

Public Preview

Welcome to the Web PubSub Service!

Use the service to build your REAL-TIME messaging applications using WebSocket APIs and the publish-subscribe pattern.

  • Connect to the service using WebSocket API from web pages, mobile apps, edge devices, etc
  • Allows up to 100K concurrent connections
  • Provide powerful APIs for you to manage these clients and deliver real-time messages

Get Started
Try Demos

Light-weighted

Clients directly use WebSocket APIs to connect and send messages.

var simpleClient = new WebSocket("wss://xxx.webpubsub.azure.com");

var subprotocolClient = new WebSocket("wss://xxx.webpubsub.azure.com", "json.webpubsub.azure.v1");
